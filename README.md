# Esp8266-Photo-Frame
Digital Photo Frame using Nodemcu Esp8266 and  1.8 Inch TFT display

  LCD     |     ESP8266
 -----------------------
 VCC            3.3V
 GND            GND
 Reset          3.3V
 A0 or D/C      D4
 SDA(MOSI)      D4
 SCK            D5
 LED            3.3V
